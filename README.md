# Java Snowflake ID

Small Java library that generates unique, time-ordered Snowflake-style IDs for distributed systems.

## Usage

Add to `pom.xml`:

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

Step 2. Add the dependency:

```xml
<dependency>
    <groupId>com.github.tenmo2003</groupId>
    <artifactId>java-snowflake-id</artifactId>
    <version>v1.0.1</version>
</dependency>
```
